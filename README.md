# cn-official-archive

本仓库用于归档和发布多个独立 HTML 页面。这些页面由公开来源收集、整理或生成，旨在以静态网页形式保存特定时间点的信息。

## 项目简介

cn-official-archive 是一个非官方的存档项目，专注于保存来自不同来源的 HTML 内容。每个页面均作为独立文件存放，不依赖任何后端服务，可直接通过浏览器或静态托管服务访问。项目不对页面内容的准确性、时效性或合法性作任何保证。

## 目录结构

```
/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的归档项目，命名方式以内容主题或日期为准。
- `assets/`：如果页面需要外部资源，请存放于此，并尽量使用相对路径引用。

## 页面归档说明

- 每个页面均为纯 HTML 格式，无动态依赖。
- 归档时尽可能保留原始页面的结构与样式，但可能因资源失效而进行调整。
- 部分页面可能来自公开存档或第三方工具生成，具体来源信息可在页面注释或对应目录内查看。
- 本仓库不存储任何用户数据，也不收集访问者信息。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或修正已有内容。

  - 内容为公开可获取的 HTML 页面。
  - 不包含任何违反法律法规或平台政策的信息。
  - 提交时请注明来源与归档日期。
